# 🌾 스마트팜 3개년 주간 데이터 자동 다운로더

이 프로그램은 사용자가 지정한 기간(예: 3개년)의 스마트팜 데이터를 **주 단위(Weekly)**로 자동 분할하여 포털 사이트(`https://jebitna.agri.jeju.kr/`)로부터 누락 없이 일괄 다운로드해주는 데스크톱 대시보드 및 자동화 도구입니다.

로컬 환경(사용자의 Mac)에서 실행되어 **보안 방해(WAF) 및 대량 호출 차단을 우회**하도록 안전하게 설계되었습니다.

---

## 🛠️ 준비 사항 (최초 1회 설정)

이 프로그램은 브라우저 자동 제어 라이브러리인 **Playwright**를 활용합니다. 터미널을 열고 다음 명령어를 실행하여 설치해 주세요.

```bash
# 1. Playwright 패키지 설치
pip install playwright

# 2. 브라우저 엔진(Chromium) 설치
playwright install chromium
```

---

## 🚀 실행 방법

1. **프로그램 폴더로 이동:**
   터미널에서 프로그램이 저장된 폴더(`weekly_downloader`)로 이동합니다.

2. **서버 및 대시보드 실행:**
   아래 명령어를 실행하여 로컬 서버를 가동합니다.
   ```bash
   python run_dashboard.py
   ```

3. **대시보드 접속:**
   서버가 기동되면 자동으로 웹 브라우저(`http://localhost:8080`)가 열리며 아래와 같은 아름다운 유리모핑 다크 대시보드가 로드됩니다.

---

## 💻 사용 방법

1. **설정값 입력:**
   * **접속 대상 URL:** `https://jebitna.agri.jeju.kr/` (기본값)
   * **포털 ID / 비밀번호:** 사용자의 로그인 정보를 입력합니다. (보안을 위해 로컬 브라우저 세션에만 임시 보관되며 서버로 전송되지 않습니다.)
   * **수집 시작일 및 종료일:** 데이터를 조회할 3개년 범위를 지정합니다. (예: `2023-01-01` ~ `2026-05-27`)
   * **다운로드 경로:** 엑셀 파일이 저장될 폴더 경로를 지정합니다. (기본값 `./downloads`)

2. **자동화 시작:**
   * `다운로드 자동화 시작` 버튼을 클릭합니다.
   * 브라우저(Chrome) 창이 자동으로 기동되어 로그인을 수행한 뒤, '동시계열' 페이지로 진입하여 주차별 데이터를 순차적으로 검색 및 다운로드합니다.

3. **진행 상황 모니터링:**
   * 대시보드 오른쪽에서 현재 몇 번째 주차(예: `[12/178]`)를 받고 있는지 실시간 게이지와 진행률(%)이 업데이트됩니다.
   * 아래 실시간 콘솔 창을 통해 동작 로그를 눈으로 즉시 확인할 수 있습니다.

## 🛡️ 안정성 및 보안 기능

* **WAF(웹 방화벽) 우회:** 헤드리스(비표시) 모드 대신 실제 브라우저를 기동하고 유저 에이전트 및 내비게이션 우회 코드를 삽입하여 클라우드 차단 정책을 통과합니다.
* **인간형 랜덤 지연 (Jitter):** 한 주가 끝날 때마다 다음 조회를 실행하기 전 3초~6초 사이의 랜덤한 대기 시간을 가져 서버 부하를 방지하고 이상 탐지를 차단합니다.
* **단 1회 세션 유지:** 매 요청마다 로그인을 새로 수행하지 않고, 최초 로그인된 세션 상태를 그대로 물고 날짜 변경만 수행하므로 매우 신속하게 다운로드됩니다.
